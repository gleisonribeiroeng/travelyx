Computed signal for filtered results:
```typescript
filteredResults = computed(() => {
  let results = this.searchResults();
  const mode = this.modeFilter();
  if (mode) {
    results = results.filter(r => r.mode === mode);
  }
  // Sort by price ascending
  return [...results].sort((a, b) => a.price.total - b.price.total);
});
```

Date getter:
- `get minDepartureDate(): Date { return new Date(); }` — no past dates

Search method (`searchTransport(): void`):
1. Return early if form invalid
2. Extract form values: origin, destination, departureDate
3. Return early if departureDate is null
4. Format date to YYYY-MM-DD string: `departureDate.toISOString().split('T')[0]`
5. Set `isSearching(true)` and `hasSearched(true)`

8. Subscribe: on result.error show snackbar "Failed to search transport options. Please try again." and set results to []; on success set results to result.data
9. Add error handler like car-search for unexpected errors

Mode icon helper (`getModeIcon(mode: string): string`):
```typescript
getModeIcon(mode: string): string {
  const icons: Record<string, string> = {
    bus: 'directions_bus',
    train: 'train',
    ferry: 'directions_boat',
    other: 'commute',
  };
  return icons[mode] || 'commute';
}
```

Duration format helper (`formatDuration(minutes: number): string`):
```typescript
formatDuration(minutes: number): string {
  const h = Math.floor(minutes / 60);
  const m = minutes % 60;
  if (h === 0) return `${m}m`;
  if (m === 0) return `${h}h`;
  return `${h}h ${m}m`;
}
```
