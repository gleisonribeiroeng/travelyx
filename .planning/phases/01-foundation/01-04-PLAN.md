**Verify angular.json fileReplacements:**

Open `angular.json` and confirm that under `projects > triply > architect > build > configurations > development`, there is a `fileReplacements` entry:


If `ng generate environments` didn't add this, add it manually to the `development` configuration. Also check if there's a `serve` configuration that extends `development` — if so, the file replacement will apply during `ng serve` automatically.
