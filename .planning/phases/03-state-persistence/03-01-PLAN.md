<task type="auto">
  <name>Task 2: Create LocalStorageService safe wrapper</name>
  <files>triply/src/app/core/services/local-storage.service.ts</files>
  <action>
Create `triply/src/app/core/services/local-storage.service.ts` as an `@Injectable({ providedIn: 'root' })` service.

Inject `MatSnackBar` from `@angular/material/snack-bar` via `inject()`.

Implement these methods:

1. **`get<T>(key: string): T | null`** — wraps `localStorage.getItem()` + `JSON.parse()` in try/catch. Returns `null` if key missing OR if JSON.parse throws (corrupt/malformed data). Never throws.

2. **`set(key: string, value: unknown): void`** — wraps `localStorage.setItem(key, JSON.stringify(value))` in try/catch. If the error is a QuotaExceededError, show a snackbar warning: `'Storage full — your trip could not be saved. Please remove some items.'` with `'Dismiss'` action, 8000ms duration, and `panelClass: 'snack-warning'`. Non-quota errors (e.g. SecurityError in private browsing) are swallowed silently — the app continues with in-memory state only.

3. **`remove(key: string): void`** — wraps `localStorage.removeItem()` in try/catch. Errors ignored.

4. **`clear(): void`** — wraps `localStorage.clear()` in try/catch. Errors ignored.

5. **`private isQuotaExceededError(err: unknown): boolean`** — cross-browser detection: checks `err instanceof DOMException` AND (`err.code === 22` OR `err.code === 1014` OR `err.name === 'QuotaExceededError'` OR `err.name === 'NS_ERROR_DOM_QUOTA_REACHED'`). This covers Chrome (code 22), Firefox (code 1014), and all modern browsers by name.

Do NOT import or use anything from `@angular/core` except `Injectable` and `inject`. Do NOT add console.log statements. The service must be fully self-contained with no dependencies on other app services.
  </action>
  <verify>Run `npx tsc --noEmit` from the triply directory. Confirm zero type errors. Verify the file exports LocalStorageService with get, set, remove, clear public methods and private isQuotaExceededError.</verify>
