Create a standalone Angular component at triply/src/app/shared/components/error-banner/error-banner.component.ts with:

**Component definition:**
- selector: 'app-error-banner'
- standalone: true
- imports: MATERIAL_IMPORTS from '../../../../core/material.exports'
- Inline template and inline styles (single-file component)

**Inputs (use Angular signal-based input API):**
- source = input.required<string>() — the API source name (e.g. "Flights", "Hotels")
- message = input.required<string>() — the error message text

**Outputs:**
- dismiss = output<void>() — emitted when user clicks the close button

**Template:**
```html
<mat-card class="error-banner">
  <mat-card-content>
    <div class="error-content">
      <mat-icon color="warn">error</mat-icon>
      <div class="error-text">
        <strong>{{ source() }} Error</strong>
        <p>{{ message() }}</p>
      </div>
      <button mat-icon-button (click)="dismiss.emit()" aria-label="Dismiss error">
        <mat-icon>close</mat-icon>
      </button>
    </div>
  </mat-card-content>
</mat-card>
```

**Styles:**
- .error-banner: background-color var(--mat-sys-error-container), color var(--mat-sys-on-error-container), margin-bottom var(--triply-spacing-md)
- .error-content: display flex, align-items flex-start, gap var(--triply-spacing-sm)
- .error-text: flex 1; strong display block margin-bottom 4px; p margin 0 font-size 0.9rem
- mat-icon[color="warn"]: color var(--mat-sys-error)
- @media (max-width: 600px): .error-content gap var(--triply-spacing-xs)

Use input() from '@angular/core' (not @Input() decorator) and output() from '@angular/core' (not @Output() decorator) following Angular 21 signal-based API pattern.
