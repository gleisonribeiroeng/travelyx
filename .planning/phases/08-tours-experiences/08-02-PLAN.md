Create tour-search.component.ts following the TransportSearchComponent pattern exactly:

1. Component decorator:
   - selector: 'app-tour-search'
   - standalone: true
   - imports: [MATERIAL_IMPORTS, ReactiveFormsModule, CommonModule]
   - templateUrl: './tour-search.component.html'
   - styleUrl: './tour-search.component.scss'

2. Exported class: TourSearchComponent

3. Inject services:
   - private readonly tourApi = inject(TourApiService)
   - private readonly tripState = inject(TripStateService)
   - private readonly snackBar = inject(MatSnackBar)

4. Reactive form (SIMPLER than transport — only destination):
   ```typescript
   tourSearchForm = new FormGroup({
     destination: new FormControl('', Validators.required),
   });
   ```

5. State signals:
   - searchResults = signal<Activity[]>([])
   - isSearching = signal(false)
   - hasSearched = signal(false)

6. NO filter signals or computed signals needed — tours don't have mode types. Sort results by price ascending in the results directly:
   ```typescript
   sortedResults = computed(() =>
     [...this.searchResults()].sort((a, b) => a.price.total - b.price.total)
   );
   ```

7. searchTours() method:
   - Guard: if (this.tourSearchForm.invalid) return
   - Extract: const destination = this.tourSearchForm.value.destination ?? ''
   - Set: isSearching(true), hasSearched(true)
   - Call: this.tourApi.searchTours({ destination })
   - Pipe: finalize(() => this.isSearching.set(false))
   - Subscribe next: check result.error — if error, snackBar.open('Failed to search tours. Please try again.', 'Close', { duration: 3000 }) and set searchResults([])
   - If no error: set searchResults(result.data)

8. addToItinerary(tour: Activity) method:
   - Call: this.tripState.addActivity({ ...tour, addedToItinerary: true })
   - Show: this.snackBar.open('Tour added to itinerary', 'Close', { duration: 3000 })

9. formatDuration(minutes: number) helper (same as transport):
   - const h = Math.floor(minutes / 60)
   - const m = minutes % 60
   - if (h === 0) return `${m}m`
   - if (m === 0) return `${h}h`
   - return `${h}h ${m}m`
   NOTE: This is only called when durationMinutes is not null. The template handles the null case with 'Flexible' text.

10. Import Activity from '../../core/models/trip.models'
11. Import TourApiService from '../../core/api/tour-api.service'
12. Import TripStateService from '../../core/services/trip-state.service'
13. Import MATERIAL_IMPORTS from '../../core/material.exports'
14. Import signal, computed, inject from '@angular/core'
15. Import MatSnackBar from '@angular/material/snack-bar'
16. Import finalize from 'rxjs/operators'

         <mat-card class="tour-card">

                 </div>
