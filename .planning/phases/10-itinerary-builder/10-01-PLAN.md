**itinerary.component.ts:**
- Inject `TripStateService` via `inject()`
- Expose `tripState.hasItems` as a readonly signal for empty state toggling
- Create a `readonly itemsByDay = computed(() => {...})` signal that:
  1. Reads `this.tripState.itineraryItems()`
  2. Groups items by `item.date` using `Object.groupBy()` (ES2024 native). If TypeScript complains about Object.groupBy not existing, use a `reduce()` fallback instead: `items.reduce((acc, item) => { if (!acc[item.date]) acc[item.date] = []; acc[item.date].push(item); return acc; }, {} as Record<string, ItineraryItem[]>)`
  3. Converts to `Map<string, ItineraryItem[]>` where each value array is sorted: null timeSlot first (all-day), then by timeSlot ascending (HH:MM strings compare lexicographically), then by `order` field ascending
  4. Returns the Map sorted by date key ascending (ISO date strings are lexicographically sortable)
  IMPORTANT: Create a shallow copy before sorting (`[...dayItems].sort(...)`) to avoid mutating the signal's internal array
- Create `moveItemUp(item: ItineraryItem, dayItems: ItineraryItem[])` method:
  1. Find the item's index in dayItems
  2. If index <= 0, return (already at top)
  3. Import and call `moveItemInArray(dayItems, index, index - 1)` from `@angular/cdk/drag-drop`
  4. Loop through all dayItems and call `this.tripState.updateItineraryItem({ ...item, order: idx })` for each to persist the new positions
- Create `moveItemDown(item: ItineraryItem, dayItems: ItineraryItem[])` with same logic but moving index to index + 1 (guard: if index >= dayItems.length - 1, return)
- Create `removeItem(id: string)` method calling `this.tripState.removeItineraryItem(id)`
- Import MATERIAL_IMPORTS from `../../core/material.exports`, ReactiveFormsModule, KeyValuePipe, DatePipe, ItineraryItemComponent (will be created in Task 2)
- Use `standalone: true`, `templateUrl`, `styleUrl`
