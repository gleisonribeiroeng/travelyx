Create a standalone form component for adding custom manual items to the itinerary.

**manual-item-form.component.ts:**
- `standalone: true`, selector `app-manual-item-form`, templateUrl, styleUrl
- Imports: MATERIAL_IMPORTS, ReactiveFormsModule
- Inject `TripStateService`, `FormBuilder`, `MatSnackBar`
- Create `readonly form` in class body (not ngOnInit) using `this.fb.group({...})`:
  - `label`: ['', Validators.required] â€” item name
  - `date`: ['', Validators.required] â€” YYYY-MM-DD via `<input type="date">`
  - `timeSlot`: [''] â€” optional, HH:MM via `<input type="time">`, same timeSlot validator from ItineraryItemComponent (returns null if empty, validates HH:MM pattern if provided)
  - `notes`: [''] â€” optional free text
- `readonly showForm = signal(false)` â€” toggle to show/hide the form (collapsed by default)
- `toggleForm()`: flips showForm signal
- `onSubmit()`:
  1. If form invalid, return
  2. Build `ItineraryItem`: `{ id: crypto.randomUUID(), type: 'custom', refId: null, date: this.form.value.date!, timeSlot: this.form.value.timeSlot || null, label: this.form.value.label!, notes: this.form.value.notes || '', order: 0 }`
  3. Call `this.tripState.addItineraryItem(newItem)`
  4. Reset form and hide it: `this.form.reset()`, `this.showForm.set(false)`
  5. Show snackbar: "Custom item added"

**manual-item-form.component.html:**
- A button to toggle the form: `<button mat-raised-button color="primary" (click)="toggleForm()"><mat-icon>add</mat-icon> Add Custom Item</button>` â€” shown when form is hidden
- `@if (showForm())` wraps the form in a `<mat-card class="manual-item-form">`
- `<form [formGroup]="form" (ngSubmit)="onSubmit()">`
- `<mat-form-field>` for label with mat-label "Item Name"
- `<mat-form-field>` for date with mat-label "Date" and `<input matInput type="date" formControlName="date" required>`
- `<mat-form-field>` for timeSlot with mat-label "Time (optional)" and `<input matInput type="time" formControlName="timeSlot">`
  - `<mat-hint>` "Leave empty for all-day item"
  - `@if (form.get('timeSlot')?.hasError('invalidTimeSlot')) { <mat-error>Must be HH:MM format</mat-error> }`
- `<mat-form-field>` for notes with mat-label "Notes (optional)" and `<textarea matInput formControlName="notes" rows="2"></textarea>`
- Actions: Save button `<button mat-raised-button color="primary" type="submit" [disabled]="form.invalid">Add</button>` and Cancel button `<button mat-button type="button" (click)="toggleForm()">Cancel</button>`

**manual-item-form.component.scss:**
- `.manual-item-form`: padding, margin-bottom, max-width for comfortable form width
- Form fields: full width within the card, stacked vertically
- Actions row: flex layout with gap
- Use `var(--triply-spacing-*)` and `var(--mat-sys-*)` tokens
