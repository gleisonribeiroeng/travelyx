1. **AmadeusFlightOffer interface** (the external API shape):
   ```typescript
   export interface AmadeusFlightOffer {
     id: string;
     itineraries: Array<{
       duration: string; // ISO 8601 duration: "PT3H35M"
       segments: Array<{
         departure: { iataCode: string; at: string }; // ISO 8601 datetime
         arrival: { iataCode: string; at: string };
         carrierCode: string;
         number: string;
         numberOfStops: number;
       }>;
     }>;
     price: {
       total: string; // String number from API
       currency: string;
     };
     validatingAirlineCodes: string[];
   }
   ```

2. **FlightMapper class** â€” `@Injectable({ providedIn: 'root' })`, implements `Mapper<AmadeusFlightOffer, Flight>`:
   - `mapResponse(raw: AmadeusFlightOffer): Flight` â€” transforms one Amadeus offer to the canonical Flight model:
     - Use FIRST itinerary (outbound only for Phase 4)
     - `origin` = first segment's departure.iataCode
     - `destination` = last segment's arrival.iataCode
     - `departureAt` = first segment's departure.at (keep ISO 8601 string as-is)
     - `arrivalAt` = last segment's arrival.at (keep ISO 8601 string as-is)
     - `airline` = validatingAirlineCodes[0]
     - `flightNumber` = `${firstSegment.carrierCode}${firstSegment.number}`
     - `durationMinutes` = parse from itinerary.duration (PT3H35M -> 215)
     - `stops` = segments.length - 1 (NOT segment.numberOfStops which is per-segment)
     - `price` = `{ total: parseFloat(raw.price.total), currency: raw.price.currency }`
     - `link` = `{ url: 'https://www.amadeus.com/booking/${raw.id}', provider: 'Amadeus' }`
     - `id` = raw.id
     - `source` = 'amadeus'
     - `addedToItinerary` = false

   - `private parseDuration(isoDuration: string): number` â€” parses ISO 8601 duration:
     - Extract hours from `(\d+)H` regex match (default 0)
     - Extract minutes from `(\d+)M` regex match (default 0)
     - Return `hours * 60 + minutes`

Import Flight from `../models/trip.models` and Mapper from `./mapper.interface`.

**Critical: Do NOT convert datetime strings to Date objects.** All datetime fields stay as ISO 8601 strings per project decision (03-01).

     returnDate?: string;   // YYYY-MM-DD (optional for one-way)

3. **FlightApiService class** â€” `@Injectable({ providedIn: 'root' })`, extends `BaseApiService`:
   - Constructor: `super('amadeus')`
   - Private fields:
     - `private readonly mapper = inject(FlightMapper)`
     - `private cachedToken: string | null = null`
     - `private tokenExpiresAt = 0`

   - **`private getAccessToken(): Observable<string>`** â€” OAuth2 token with caching:
     - If `cachedToken` exists AND `tokenExpiresAt > Date.now()` â†’ return `of(this.cachedToken)`
     - Otherwise, POST to `/v1/security/oauth2/token` with body `grant_type=client_credentials&client_id=${apiKey}&client_secret=${apiSecret}`
     - Use `this.http.post()` directly (NOT `this.post()`) because the token endpoint needs `Content-Type: application/x-www-form-urlencoded`, not JSON
     - Set `context: new HttpContext().set(API_SOURCE, 'amadeus')` on the request
     - Read API key and secret from environment: `environment.amadeusApiKey` and `environment.amadeusApiSecret`
     - On success: cache `access_token`, set `tokenExpiresAt = Date.now() + (expires_in - 120) * 1000` (2-minute buffer before actual expiry)
     - Return the token string via `tap()` + `map()`
     - Import environment from `../../../environments/environment`

   - **`private authenticatedGet<T>(path: string, params?: Record<string, string | number | boolean>): Observable<T>`** â€” wraps get() with bearer token:
     - Call `getAccessToken()` via `switchMap`
     - Then call `this.http.get<T>(url, { params, headers: { Authorization: 'Bearer ' + token }, context })` â€” note: must manually build URL and context like `BaseApiService.get()` does, because we need to add the Authorization header
     - Apply `withBackoff()` for rate limit retry on the inner observable

   - **`searchFlights(params: FlightSearchParams): Observable<ApiResult<Flight[]>>`**:
     - Call `authenticatedGet<{ data: AmadeusFlightOffer[] }>('/v2/shopping/flight-offers', { originLocationCode: params.origin, destinationLocationCode: params.destination, departureDate: params.departureDate, ...(params.returnDate && { returnDate: params.returnDate }), adults: params.adults, max: 50 })`
     - Pipe through `map(response => ({ data: response.data.map(offer => this.mapper.mapResponse(offer)), error: null as null }))`
     - `catchError(error => of({ data: [] as Flight[], error: error as AppError }))`
     - Return type is `Observable<ApiResult<Flight[]>>`

   - **`searchAirports(keyword: string): Observable<AirportOption[]>`**:
     - Guard: if `keyword.length < 2` return `of([])`
     - Call `authenticatedGet<{ data: Array<{ iataCode: string; name: string; address?: { cityName?: string } }> }>('/v1/reference-data/locations', { keyword, subType: 'AIRPORT,CITY' })`
     - Map response: `response.data.map(airport => ({ iataCode: airport.iataCode, name: airport.name, cityName: airport.address?.cityName || airport.name }))`
     - `catchError(() => of([]))` â€” airport autocomplete must never surface errors to user
