Purpose: Without this, every HTTP request from the Angular dev server to an external API will fail with CORS errors. The dev proxy is the development counterpart to the production CORS proxy (Cloudflare Worker / Netlify Edge Function) that will be set up when deploying.

Output: proxy.conf.json with entries for all 6 API categories, angular.json updated with proxyConfig option.
</objective>

  <name>Task 1: Create proxy.conf.json with entries for all API targets</name>
  <files>triply/src/proxy.conf.json</files>
  <action>
Create `triply/src/proxy.conf.json` with proxy entries for all 6 API categories plus Google Places. Each entry maps the proxy-relative path (matching ApiConfigService.endpoints) to the actual external API base URL.

Configure these entries:

1. `/api/amadeus` -> `https://test.api.amadeus.com`
   - `changeOrigin: true`, `pathRewrite: { "^/api/amadeus": "" }`, `secure: true`
   - Note: In production, the Amadeus proxy will also handle OAuth2 token exchange. In dev, the proxy just forwards — actual Amadeus auth testing requires valid credentials.

2. `/api/hotels` -> `https://booking-com15.p.rapidapi.com`
   - `changeOrigin: true`, `pathRewrite: { "^/api/hotels": "" }`, `secure: true`

3. `/api/cars` -> `https://booking-com15.p.rapidapi.com`
   - `changeOrigin: true`, `pathRewrite: { "^/api/cars": "" }`, `secure: true`
   - Note: Using same RapidAPI host as hotels — path differentiation happens in the service layer.

4. `/api/transport` -> `https://api.example.com`
   - `changeOrigin: true`, `pathRewrite: { "^/api/transport": "" }`, `secure: true`
   - Note: Rome2rio is unavailable. Use placeholder target `https://api.example.com`. The transport API provider is TBD — this entry ensures the proxy pattern works; the actual target will be updated when a transport API is selected.

5. `/api/tours` -> `https://api.viator.com`
   - `changeOrigin: true`, `pathRewrite: { "^/api/tours": "" }`, `secure: true`

6. `/api/attractions` -> `https://api.opentripmap.com`
   - `changeOrigin: true`, `pathRewrite: { "^/api/attractions": "" }`, `secure: true`

7. `/api/places` -> `https://maps.googleapis.com`
   - `changeOrigin: true`, `pathRewrite: { "^/api/places": "" }`, `secure: true`

All entries use `"logLevel": "debug"` so proxy activity is visible in the dev server console during development.

  <verify>Verify the file is valid JSON by parsing it: `node -e "JSON.parse(require('fs').readFileSync('triply/src/proxy.conf.json','utf8')); console.log('Valid JSON')"`. Verify all 7 entries exist.</verify>
