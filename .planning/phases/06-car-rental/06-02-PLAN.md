**Step 1: Create CarSearchComponent TypeScript**

Create `triply/src/app/features/car-search/car-search.component.ts`:

```typescript
@Component({
  selector: 'app-car-search',
  standalone: true,
  imports: [MATERIAL_IMPORTS, ReactiveFormsModule, CommonModule],
  templateUrl: './car-search.component.html',
  styleUrl: './car-search.component.scss',
})
export class CarSearchComponent { ... }
```

1. **Injected services:**
   - `private readonly carApi = inject(CarApiService)`
   - `private readonly tripState = inject(TripStateService)`
   - `private readonly snackBar = inject(MatSnackBar)`

2. **Form controls (reactive form with FormGroup):**
   ```typescript
   carSearchForm = new FormGroup({
     pickupLocation: new FormControl('', Validators.required),
     dropoffLocation: new FormControl('', Validators.required),
     pickupDate: new FormControl<Date | null>(null, Validators.required),
     pickupTime: new FormControl('10:00', Validators.required),
     dropoffDate: new FormControl<Date | null>(null, Validators.required),
     dropoffTime: new FormControl('10:00', Validators.required),
     driverAge: new FormControl(30, [Validators.required, Validators.min(18), Validators.max(99)]),
   });
   ```
   NOTE: Unlike hotels which use autocomplete with DestinationOption objects, cars use plain text inputs for locations per user decision. This is simpler and avoids overengineering for an unverified API. The text value is passed directly to the API as pickupLocation/dropoffLocation.

3. **Filter signals (separate from form, for client-side filtering):**
   - `vehicleTypeFilter = signal<string>('')` -- empty string means "All Types"
   - `maxPriceFilter = signal<number | null>(null)` -- null means no max price filter

4. **Vehicle type options (for the filter dropdown):**
   ```typescript
   vehicleTypes: string[] = ['Economy', 'Compact', 'Intermediate', 'Full Size', 'SUV', 'Premium', 'Minivan'];
   ```

5. **Search state signals:**
   - `searchResults = signal<CarRental[]>([])`
   - `isSearching = signal(false)`
   - `hasSearched = signal(false)`

6. **Computed signal for filtered and sorted results:**
   ```typescript
   filteredCars = computed(() => {
     let results = this.searchResults();
     const typeFilter = this.vehicleTypeFilter();
     const maxPrice = this.maxPriceFilter();

     if (typeFilter) {
       results = results.filter(car =>
         car.vehicleType.toLowerCase().includes(typeFilter.toLowerCase())
       );
     }

     if (maxPrice !== null && maxPrice > 0) {
       results = results.filter(car => car.price.total <= maxPrice);
     }

     // Sort by price ascending (lowest first)
     return [...results].sort((a, b) => a.price.total - b.price.total);
   });
   ```

7. **Date getters:**
   - `get minPickupDate(): Date { return new Date(); }`
   - `get minDropoffDate(): Date { return this.carSearchForm.value.pickupDate || new Date(); }`

8. **Methods:**
   - `searchCars(): void` -- validates form, builds CarSearchParams, calls carApi.searchCars():
     - Extract form values
     - Build ISO 8601 datetimes by combining date + time:
       ```typescript
       const pickupDateStr = pickupDate.toISOString().split('T')[0]; // YYYY-MM-DD
       const pickupAt = `${pickupDateStr}T${pickupTime}:00`;         // YYYY-MM-DDTHH:MM:00
       const dropoffDateStr = dropoffDate.toISOString().split('T')[0];
       const dropoffAt = `${dropoffDateStr}T${dropoffTime}:00`;
       ```
     - Call `this.carApi.searchCars({ pickupLocation, dropoffLocation, pickupAt, dropoffAt, driverAge })`
     - Set `isSearching(true)` before call, use `finalize(() => this.isSearching.set(false))`
     - Set `hasSearched(true)`
     - On success with error: show snackbar "Failed to search car rentals. Please try again.", set results to []
     - On success without error: set searchResults to result.data
     - On error (subscribe error handler): show snackbar "An error occurred. Please try again.", set results to []

   - `addToItinerary(car: CarRental): void` -- calls `this.tripState.addCarRental({ ...car, addedToItinerary: true })`, shows snackbar "Car rental added to itinerary"

   - `setVehicleTypeFilter(value: string): void` -- sets vehicleTypeFilter signal

   - `setMaxPriceFilter(value: string): void` -- parses string to number, sets maxPriceFilter signal:
     ```typescript
     const num = parseFloat(value);
     this.maxPriceFilter.set(isNaN(num) || num <= 0 ? null : num);
     ```

   - `formatPrice(car: CarRental): string` -- returns formatted price for display (use Angular currency pipe in template instead, but have this as backup)

Import from:
- `@angular/core`: Component, signal, computed, inject
- `@angular/forms`: FormControl, FormGroup, Validators, ReactiveFormsModule
- `@angular/common`: CommonModule
- `@angular/material/snack-bar`: MatSnackBar
- `../../core/material.exports`: MATERIAL_IMPORTS
- `../../core/api/car-api.service`: CarApiService, CarSearchParams
- `../../core/services/trip-state.service`: TripStateService

- RxJS: finalize
